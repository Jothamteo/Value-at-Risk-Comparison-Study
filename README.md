# Value-at-Risk-Comparison-Study
A short comparison among the different Value-at-Risk methods, namely Parametric VaR, Parametric EWMA VaR, Historical Simulation and Filtered Historical Simulation.
In this simple example, we will illustrate VaR calculation in Python, both for single value & VaR series in time. Data utilized in this illustration is the SPY downloaded from https://www.nasdaq.com/symbol/spy/historical.
